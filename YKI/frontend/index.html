🛡️ Proje Hakkında

Bu yazılım, TEKNOFEST Çelik Kubbe Hava Savunma Sistemi yarışması için geliştirilmiş açık kaynaklı bir yer kontrol istasyonudur.

QGroundControl'den ilham alınarak, RTSP kamera stream, gerçek zamanlı telemetri ve YOLO tabanlı hedef takip görüntüleme özellikleri sunmaktadır.

⚙️ Teknik Bilgiler

FrontendHTML5 + Vanilla JS
BackendNode.js + Express + WS
VideoFFmpeg + JSMpeg (MPEG-1)
TelemetriUDP / JSON
KomutTCP / JSON
AraçRaspberry Pi 5 + ESP32
